# L10n Debt — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task.
> **Delegate to:** Gemini CLI (`gemini -p "..." --yolo --skip-trust`)

**Goal:** Move all hardcoded Portuguese/English strings in AI chat, Settings, Pantry, Shopping List and Meal Planner to `AppLocalizations` (86 locales).

**Architecture:** Add ARB keys to `lib/l10n/app_en.arb`, run `flutter gen-l10n`, then replace every hardcoded string with the corresponding `l10n.*` call. Touches NO logic — pure string replacement.

**Tech Stack:** Flutter l10n (`flutter gen-l10n`), `AppLocalizations.of(context)!`, `lib/l10n/app_en.arb`

**Rules:**
- After EVERY file edit, run: `flutter analyze --fatal-infos lib/app/<feature>/`
- After ALL ARB edits, run: `flutter gen-l10n` before using any new key
- Do NOT refactor anything else. String replacement only.
- `always_put_control_body_on_new_line` lint is active.
