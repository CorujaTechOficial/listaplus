Mudancas por secao

AppBar (novo)

  • Nome da lista + subtitulo "Assistente de compras"
  • Energia IA como badge com barra + numero
  • Trocar lista (⇄) e menu (⋮) com historico, busca, limpar
  • Remove o _HeaderRow interno do AiChatPanel

List Hero Card

  • Card com gradiente verde, progresso e custo
  • Colapsa em pill ao scrollar (3/8 + custo)
  • Expande ao tocar (substitui _CompactListCard)
  • Modo Mercado mantido

Chat Bubbles

  • IA com avatar circular + nome acima
  • Acoes (adicionar, organizar) fora do bubble
  • Feedback unificado: 1 linha (hora, like, dislike, copy, regenerar)
  • Remove feedback duplicado dentro do bubble

Empty State + Input

  • Orb menor (64px), titulo + subtitulo
  • Apenas 2 sugestoes (era 3)
  • Input com border-radius maior (pill)
  • Shortcut chips acima do input mantidos
